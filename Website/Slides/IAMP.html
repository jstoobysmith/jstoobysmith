A published paper, and a chain of comments and replies arguing over whether a pair of terms (B₁ + B₂) belongs in the result — in, then out, then back in. Even careful, peer-reviewed physics is hard to be sure of.

… and one of my own papers

It happens to all of us — including me. The point is not blame; it is that there is currently no scalable way to be sure a result is correct.

"Erdős Problem #728 — the first Erdős problem resolved fully autonomously by an AI system, produced directly as a formal proof in Lean."

— GPT-5.2 Pro (OpenAI) + Aristotle (Harmonic), January 2026

arXiv:2601.07421 — Resolution of Erdős Problem #728: a writeup of Aristotle's Lean proof

January 2026 — the first Erdős problem solved fully autonomously by an AI system.
The system: a combination of GPT-5.2 Pro (OpenAI) and Aristotle (Harmonic), operated by Kevin Barreto. It generated the proof directly in Lean; Nat Sothanaphan then wrote up an informal version (arXiv:2601.07421).
The statement: for any 0 < C₁ < C₂ and 0 < ε < 1/2, there are infinitely many triples (a, b, n) with a!·b! dividing n!·(a+b−n)! and C₁ log n < a+b−n < C₂ log n.
The trust point: because the output is a machine-checked Lean proof, it is a result, not a claim — contrast October 2025, when "solved" Erdős problems turned out to already be in the literature.
